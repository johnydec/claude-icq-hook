The sound you
never forgot.

Plays the classic ICQ "Uh Oh!" after every Claude Code response. One command. Pure nostalgia.

npx claude-icq-hook
Step 01

Run the command

One npx command copies the sound and sets up the hook.

Step 02

Restart Claude Code

The hook gets added to ~/.claude/settings.json automatically.

Step 03

Feel 1996 again

Every time Claude finishes a response — Uh Oh!
